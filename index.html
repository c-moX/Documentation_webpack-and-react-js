# Webpack.org documentation!

* Google classroom || Class: 06 & 07

Step: 01

For 'Package.json' file ___

Install the "CMder" terminal and target folder location, then type below code...

npm init -y

Step: 02

Then type for 'webpack cli' dependencise...

npm i -D webpack webpack-cli

Step: 03

Against, create the | folder & file | in your directory...

  1. src (folder)
  2. index.js
  3. index.html
  4. webpack.config.js
Note: please! link to the webpack.config.js file on browser support link 'webpack.org' entry/output section (copy_past).

Then type CMder terminal on 'development mode' ...

npm run build

Step: 04

ADD to the 'dependencise' and 'babel loder/pre-set' in (package.json & webpack.config.js) files.

Install to below code...

npm install -D babel-loader @babel/core @babel/preset-env webpack

Note: please! link to the webpack.config.js file on browser support link 'webpack.org' babel-loder section (copy_past).
